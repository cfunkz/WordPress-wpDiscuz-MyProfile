## Profile Insights Pro

User dashboard plugin for WordPress with Karma tracking, wpDiscuz subscriptions, ratings, notifications, author statistics, and front-end profile editing.

### Installation

1. Download the **.zip** from the GitHub releases page:
   [Release with .zip file](https://github.com/cfunkz/WordPress-wpDiscuz-MyProfile/releases)

2. In WordPress Admin → **Plugins → Add New → Upload Plugin**

3. Upload the ZIP file and activate.

4. Create a new page and add the shortcode:

```text
[user_insights]
```

### Custom Styling

You can modify the layout inside:

**Admin → Profile Insights → Custom CSS**

Example:

```css
.dpi-wrap {
    max-width: 900px;
}
```

Custom CSS is appended after the plugin’s default styles and safely sanitised.
